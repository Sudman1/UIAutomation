# Get-UIRoot

## SYNOPSIS
Retrieve the Root UI Element.

## SYNTAX

```
Get-UIRoot [<CommonParameters>]
```

## DESCRIPTION
The root element represents the current desktop and its child elements represent application windows.

## EXAMPLES

### EXAMPLE 1
```
Get-UIRoot
```

Returns the root element
